# Website CV Pribadi - UTS Pemrograman Web I

Project ini dibuat untuk memenuhi tugas Ujian Tengah Semester mata kuliah **Pemrograman Web I**. Website ini berisi halaman Curriculum Vitae (CV) pribadi yang dibuat menggunakan HTML, CSS, dan JavaScript.

## Deskripsi Project

Website ini menampilkan informasi CV pribadi yang terdiri dari profil singkat, deskripsi diri, riwayat pendidikan, kemampuan, hobi, galeri gambar, link terkait, dan kontak. Project ini dibuat sebagai latihan penerapan dasar-dasar pemrograman web, khususnya penggunaan HTML, CSS, dan JavaScript.

## Fitur Website

Beberapa fitur dan elemen yang terdapat pada website ini antara lain:

* Halaman profil pribadi
* Foto diri
* Deskripsi diri dalam bentuk paragraf
* Heading menggunakan tag HTML
* Komentar HTML pada struktur kode
* Text formatting seperti `strong`, `em`, dan `mark`
* Link internal dan eksternal
* Galeri gambar
* Tabel riwayat pendidikan
* Tabel kemampuan atau skill
* List hobi dan data singkat
* Tombol interaktif sederhana menggunakan JavaScript
* Tampilan responsive untuk desktop dan mobile

## Teknologi yang Digunakan
